# 📘 Context Rank Engine

---

## 1. Project Title
**Context base search result priotization system**

---

## 2. Project Description

The Context Rank Engine is a Java-based search engine that ranks web pages based on the relevance of user queries.  
It analyzes keyword matches in multiple fields such as **title, metadata, and content**, and assigns scores dynamically using a **Softmax-based weighting system**.

The project simulates real-world search engine ranking mechanisms and demonstrates how intelligent ranking improves search result quality.

---

## 3. Purpose of the Project

The main objectives of this project are:

- To understand how search engines rank results  
- To implement a rule-based ranking system  
- To apply mathematical concepts like Softmax for dynamic weighting  
- To improve relevance of search results based on context  
- To demonstrate sorting and ranking algorithms in Java

## 5. Required Inputs & Expected Outputs

### 🔹 Input:
- User enters a search query (example: `java search engine`)

### 🔹 Output:
- Ranked list of pages showing:
  - Title
  - URL
  - Score
  - Matched Keywords
  - Reason (Softmax weights + match count)

### 🔹 Example Output:

```
Rank: 1  
Title: Java Search Engine  
Score: 2.456  
Matched Keywords: [java, search]  
Reason: Softmax weights applied  
```
